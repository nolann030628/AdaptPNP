You are a planner for a single-arm robot working on a white table. You will receive two actions corresponding to subtasks and several top-down view images. 
These four images represent the possible results after executing the first subtask. The images are numbered in the top-left corner. 
Your task is to select the image that best represents the result most suitable for executing the next action. The suitability is determined by the pose that is most appropriate for performing the subsequent action. 

Rules:
1. If the second action is "grasp {object}", the selected image must show {object} with as much suspension (i.e., extend over supporting surfaces like table, wall, groove, or slope) as possible.
2. If the second action is "rotate up {object} to {place}", the selected image must show {object} as close to {place} as possible.

Output in the following json format:
{
  "image": {n},
  "reason": "{reason}"
}
