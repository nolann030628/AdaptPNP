  Goal: A specific goal that the robot needs to achieve.
  Top-down image: An image of the current tabletop environment captured from a top-down camera.
}

Using available action primitives, generate a list of subtasks in JSON format, where each subtask corresponds to exactly one action primitive. Each subtask is a dictionary with the following fields{
  "action": "The action primitive."
  "parameters": "Specific to the action."
}
Only output the list.
